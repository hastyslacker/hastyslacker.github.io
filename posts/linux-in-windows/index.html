Linux in Windows

Posted on Feb 25, 2022

Escrevo estas linhas do arquivo linux-in-windows.md utilizando o Vim e em breve elas serão transformadas pelo HUGO para um formato HTML e subidas para um repósito github para serem espalhadas pela internet através de um git push escrito num script bash.

Nada disso é propriamente uma novidade. Faço este processo desde que comecei a povoar este cantinho da internet com conteúdo. A diferença é que agora estou fazendo isso da minha máquina que roda Windows 10. Mas como usar essa ferramentas usuais de linux no Windows? Bem, através do Windows Subsystem for Linux.

Após habilitar o WSL, é extremamente fácil selecionar e baixar uma das distros disponíveis. No caso, estou usando Debian. Utilizando a ferramenta ´apt´ instalei os pacotes necessários, clonei os repositórios da internet e comecei a gerar novos arquivos já do windows.

Pro meu caso, não é que seja especialmente prático. A única vantagem que tenho com isso é que posso ficar online no Microsoft Teams fingindo que estou trabalhando. Por falar nisso, deixa eu ir que tenho uma reunião inútil para participar.
