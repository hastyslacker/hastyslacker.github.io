Not Standard Apt

Posted on Feb 7, 2021

Tenho tentando usar a distribuição Pop!_OS tanto quanto possível. Trata-se de uma distribuição do GNU/Linux baseada na Ubuntu e feita pela empresa System76. Não que eu desgoste do Windows. Na verdade até tenho achado o Windows 10 um sistema operacional bem bom - nem me lembro a última vez que tive um problema com ele. Mas as possibilidades que uma distribuição Linux oferece para o usuário que gosta de fuçar são infinitas e, além disso, o suporte da comunidade (vídeos no YouTube, posts de blog, etc) é algo notável.

Nas vezes que eu programei em Python no Windows, costumava utilizar a IDE Spyder, que vinha no Anaconda. Mas, pra falar a verdade, nunca gostei muito da ideia do Anaconda - se eu quero usar apenas o Spyder, porque teria que instalar RStudio, Jupyter Notebooks e outros softwares que acompanham o pacote ofídio?

Daí que no Pop!_OS eu instalei a versão standalone do Spyder através do gerenciador de pacotes do sistema (utilizando o bom e velho comando sudo apt-get install spyder). Ocorre que a versão do Spyder nos repositórios que constam na sources.list que vem configurada no Pop!_OS é a 3.3.3 - a útlima versão do software que entrou no releases estáveis. Para baixar a versão mais atual do programa (a 4.2.1), temos que sair do caminho usual.

Primeiramente precisamos adicionar a seguinte linha ao arquivo /etc/apt/sources.list: deb http://ftp.br.debian.org/debian sid main. Isso faz com que o apt possa buscar no repositório do link os pacotes que estão na versão unstable (sid).

sid é uma fonte não-assinada, de modo que quando vamos fazer o sudo apt-get update --allow-insecure-repositories para que o apt consiga atualizar a lista de pacotes disponíveis incluindo os que não estão nos repositórios inseguros.

Depois disso, quando vamos instalar de fato o software utilizamos sudo apt-get -t sid install spyder. A opção -t indica que a instalação deve ser feita a partir do binário contido no repositório inseguro.

Voilá: spyder 4.2.1 instalado. Espero que não quebre nada no meu computador.
